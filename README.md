# Portafolio Interactivo — Pablo Sanchez Abarca

Sitio web de portafolio personal y profesional con formato de **minijuego espacial**.
En lugar de secciones estáticas, navegas por el espacio con una nave y visitas
planetas que contienen mi información (perfil, experiencia, proyectos, etc.).
Pensado para que reclutadores y visitantes conozcan mi trabajo de forma interactiva.

## ✨ Características

- **Navegación espacial**: muévete con `WASD` o las flechas del teclado.
- **Planetas de información**: cada planeta es una sección (Perfil, Experiencia,
  Proyectos, Habilidades, Educación, Certificaciones, Contacto). Acércate y presiona
  `E` para abrirla.
- **Radar**: viaja directo a cualquier sección desde el panel de la izquierda.
- **Asteroides + láser**: es interactivo; dispara con `Z` para destruir asteroides.
- **Chatbot local (Ollama)**: en el "Sol de Proyectos" hay un chat conectado a un
  modelo local de Ollama con contexto profesional.
- **Bilingüe**: español e inglés, seleccionable al inicio.
- **Responsive**: controles táctiles para móvil.

## 🎮 Controles

| Acción | Teclado | Móvil |
|---|---|---|
| Mover | `W` `A` `S` `D` / Flechas | D-pad |
| Interactuar con planeta | `E` / `Espacio` | Botón `E` |
| Disparar láser | `Z` | Botón `Z` |
| Cerrar panel | `Esc` | Botón `X` |

## 🛠️ Tecnologías

- HTML5 + CSS3
- JavaScript (Canvas 2D, sin dependencias ni frameworks)
- Integración opcional con [Ollama](https://ollama.com/) para el chatbot local

## 🚀 Uso

No requiere instalación ni build. Abre `index.html` en el navegador.

Para probar el chatbot del "Sol de Proyectos" necesitas tener Ollama corriendo
localmente (por defecto `http://localhost:11434` con el modelo `llama3.2:3b`).
